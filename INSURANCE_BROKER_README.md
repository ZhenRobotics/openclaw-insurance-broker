## 📋 项目简介

这是一个为 OpenClaw 平台开发的智能保险经纪人 Skill，可以通过 WhatsApp、Telegram、Slack、Discord 等任何消息渠道提供专业的保险咨询和服务。

### 核心价值

- 💰 **节省成本** - 免费提供专业保险咨询，节省传统经纪人佣金
- ⚡ **即时响应** - 24/7 随时可用，即时回答保险问题
- 🎯 **客观中立** - 基于用户需求推荐，不受佣金影响
- 📊 **数据驱动** - 基于真实产品数据和市场信息
- 💬 **易于使用** - 自然语言交互，零学习成本

## 🎯 核心功能（8 大模块）

### 1. 🔍 保险需求分析

**功能描述**：
- 通过多轮对话收集用户信息
- 分析家庭保障缺口
- 评估风险等级
- 生成需求分析报告

**使用示例**：
```
用户: "我需要买保险，但不知道买什么"
系统: 我来帮您分析一下保险需求...
      • 请问您的年龄？
      • 职业是什么？
      • 家庭成员情况？
      ...
```

---

### 2. 📦 保险产品推荐

**功能描述**：
- 基于需求分析智能推荐产品
- 覆盖健康险、寿险、意外险、财产险
- 多维度匹配（年龄、预算、保障需求）
- 推荐 3-5 款最合适的产品

**产品类型**：
- 🏥 **健康险**：重疾险、医疗险、防癌险
- 💼 **寿险**：定期寿险、终身寿险、两全险
- 🚑 **意外险**：综合意外险、交通意外险
- 🏠 **财产险**：车险、家财险、责任险

**使用示例**：
```
用户: "推荐适合 30 岁、预算 5000 元的保险"
系统: 🎯 为您推荐以下保险产品

      1. XX 重疾险 - 保额 50 万
         匹配度: 95% ⭐⭐⭐⭐⭐
         年保费: 3,500 元
         ...
```

---

### 3. 💼 保险方案设计

**功能描述**：
- 设计个性化保险组合方案
- 考虑预算和保障需求
- 计算合理的保额配置
- 生成多套方案对比

**使用示例**：
```
用户: "帮我设计一套完整的保险方案"
系统: 💼 为您设计 3 套保险方案

      方案一：基础保障型（预算 5000 元/年）
      • 重疾险 50 万
      • 医疗险 300 万
      • 意外险 100 万
      ...
```

---

### 4. 🔎 保单对比分析

**功能描述**：
- 横向对比不同保险产品
- 保障内容详细对比
- 价格和性价比分析
- 优劣势分析

**使用示例**：
```
用户: "对比平安福和国寿福"
系统: 📊 产品对比分析

      | 对比项 | 平安福 | 国寿福 |
      |--------|--------|--------|
      | 保费   | 5,000  | 4,500  |
      | 保额   | 50万   | 50万   |
      ...
```

---

### 5. 📋 理赔协助

**功能描述**：
- 理赔流程详细指导
- 理赔材料准备清单
- 常见理赔问题解答
- 理赔注意事项提醒

**使用示例**：
```
用户: "得了重疾，如何理赔？"
系统: 📋 重疾险理赔指南

      理赔流程（5 个步骤）：
      1. 报案（48 小时内）
      2. 准备材料
      3. 提交理赔申请
      ...
```

---

### 6. 💡 保险知识解答

**功能描述**：
- 保险术语和概念解释
- 投保注意事项指导
- 健康告知填写建议
- 免责条款解读

**使用示例**：
```
用户: "什么是等待期？"
系统: 💡 等待期（观察期）

      定义：从保单生效日起，保险公司不承担
            保险责任的一段时间。

      目的：防止逆选择（带病投保）
      ...
```

---

### 7. 💰 保费计算

**功能描述**：
- 在线保费快速估算
- 不同年龄和保额的费率对比
- 保费预算建议
- 性价比分析

**使用示例**：
```
用户: "30 岁买 50 万重疾险要多少钱？"
系统: 💰 保费估算

      30 岁，50 万保额，保至 70 岁：
      • 预估年保费：3,000 - 4,000 元
      • 月均：250 - 333 元
      ...
```

### 8. 📅 保单管理

**功能描述**：
- 保单信息记录和查询

- 定期保单检视
- 保障缺口分析

**使用示例**：
```
用户: "查看我的所有保单"
系统: 📅 您的保单列表

      1. XX 重疾险
         保额: 50 万
         年保费: 3,500 元
         到期日: 2025-12-31
      ...
```

## 🚀 快速开始

### 安装部署

```bash
# 1. 进入项目目录
cd /home/justin/ai-insurance

# 2. 安装依赖
pnpm install

# 3. 构建项目
pnpm build

# 4. 一键部署
bash scripts/setup.sh
```

### 开始使用

在任何已连接的消息渠道（WhatsApp/Telegram/Slack 等）发送：

```
"我想买保险"
"推荐健康险"
"帮我分析保险需求"
"计算保费"
"如何理赔？"
```
