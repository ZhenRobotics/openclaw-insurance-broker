## 🏗️ 技术架构

### 核心架构
```
Multi-Agent 架构
├── BP Agent (商业计划书)
├── Financial Agent (财务建模)
├── Investor Match Agent (投资人匹配)
└── Strategy Agent (融资策略)

每个 Agent 独立负责一个功能模块
通过 Intent Parser 智能路由
```

### 技术栈
| 层级 | 技术 |
|------|------|
| **运行时** | Node.js >= 22 |
| **语言** | TypeScript 5.3+ |
| **平台** | OpenClaw |
| **AI** | Claude (via OpenClaw) |
| **工具** | pnpm, ESLint, Prettier |

### 关键特性
- ✅ 100% TypeScript 类型安全
- ✅ 模块化设计，易于扩展
- ✅ 智能意图识别
- ✅ 多轮对话支持
- ✅ 会话状态管理

## 🚀 部署方式

### 一键部署（推荐）
```bash
cd /home/justin/ai-insurance
bash scripts/setup.sh
```

### 手动部署
```bash
# 1. 安装依赖
pnpm install

# 2. 构建项目
pnpm build

# 3. 链接到 OpenClaw
ln -s $(pwd) ~/.openclaw/skills/fa

# 4. 配置 OpenClaw
# 编辑 ~/.openclaw/config.yaml 添加配置

# 5. 重启 Gateway
openclaw gateway restart
```

### 验证安装
```bash
# 检查 skill 状态
openclaw skill list

# 发送测试消息（在任何已连接的渠道）
"帮我生成一份商业计划书"

# 查看日志
openclaw logs --follow
```
